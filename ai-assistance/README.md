# AI Assistance

This folder contains the AI-related artifacts used during the assignment,
as requested.

- `conversations.zip`: Export of the AI conversations used during development.
- `skills/plan`: A skill I use to ask the AI to outline a plan before generating
  code or solutions. This helps me validate the approach, iterate, and keep
  ownership over technical decisions.

AI was used as a support tool for reasoning and iteration. All design decisions,
implementation, and final code were written and reviewed by me.
